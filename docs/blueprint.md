# **App Name**: Arvage AniTools

## Core Features:

- AI Story Idea Generator: Utilize a generative AI tool to suggest plotlines, character concepts, or scene outlines based on user input for 2D animation projects.
- Asset Management Library: Upload, categorize, and manage 2D characters, props, and background images in a web-based asset library for easy access and reuse.
- Static Scene Composer: A drag-and-drop interface for users to combine existing assets, set backgrounds, and arrange elements to build static 2D scenes for animation pre-visualization.
- Web-Based Cartoon Stylizer: An AI-powered feature where users can upload still images or short video clips to be processed and converted into various 2D cartoon-like visual styles, providing downloadable results.
- Project Overview and Export: View and manage all composed scenes and processed assets for a project, with options to export individual elements or simple image sequences.

## Style Guidelines:

- The application employs a dark theme. The primary interactive color is a vibrant violet (#6619FF), symbolizing creativity and digital craft. The background uses a heavily desaturated violet-grey (#1D1820) to ensure a comfortable working environment for prolonged periods. An accent blue (#8BB0FF) provides contrast for key interface elements and notifications.
- Headlines and prominent UI elements use 'Space Grotesk' (sans-serif) for a modern, digital aesthetic. Body text and longer descriptions use 'Inter' (sans-serif) for clear readability across tools and information panels.
- Line-based, minimalistic icons are used throughout the interface to clearly represent tool functionalities and navigation elements, maintaining a clean and modern appearance suitable for a creative studio.
- The layout features a central workspace for scene composition, flanked by customizable side panels for asset libraries and property inspectors. A compact timeline or project overview panel resides at the bottom.
- Subtle and functional animations are used for state changes, hover effects on interactive elements, and transitions between tool modes, enhancing user feedback without being distracting.
